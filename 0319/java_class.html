Class

함수를 포장하는 기술, 객체의 탬플릿

클래스의 기본단위 - 생성자와 메소드(함수)

클래스명은 대문자로 시작

객체 단위로 코드를 그룹화하고 재사용하기 위해 사용한다.

  • 객체

    자바스크립트는 개체기반언어이고 자바스크립트를 이루고 있는 모든 것이 객체이다.

    정확히는 원시타입을 제외한 모든 것

    객체는 프로퍼티로 구성

  • 인스턴스

    클래스를 사용하려면 일반적으로 인스턴스를 생성해서 사용

    한 페이지 내에 두 개 이상의 같은 동작을 하는 ui를 만들경우 두 개의 클래스를 만드는 것이 아니라 하나의 클래스에 두 개의 인스턴스를 만들어 사용한다.

  • 프로퍼티

    클래스 내부에 만드는 변수 (멤버면수)

    프로퍼티는 key와 value로 구성된다.

    프로퍼티가 함수일 경우 메소드라 한다.

  • 메소드

    클래스 내부에 만드는 함수

  • object의 생성자 함수
